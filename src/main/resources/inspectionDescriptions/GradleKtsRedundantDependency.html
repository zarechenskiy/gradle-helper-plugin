Reports libraries that were declared as an implementation dependency in one of the parent modules.

Example:

Parent module build.gradle.kts file: 

    dependencies {
        implementation("com.google.code.gson:gson:2.8.8")
    }
Child module build.gradle.kts file: 

    dependencies {
        implementation("com.google.code.gson:gson:2.8.8")
    }

After the quick-fix is applied child module's build.gradle.kts file will look like:


    dependencies {

    }
